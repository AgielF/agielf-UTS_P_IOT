## Instalasi
1. Clone repositori ini:
   ```bash
   git clone https://github.com/username/proyek-anda.git
   ```
2. Instal dependensi:
   ```bash
   cd proyek-anda
   npm run dev
   ```
3. Setup database (buat tabel yang diperlukan di MySQL).
4. Jalankan server:
   ```bash
   npm run dev
   ```
5. Buka aplikasi di browser: `http://localhost:3000`
